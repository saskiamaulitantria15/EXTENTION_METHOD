Review sesi 8 - Builder Pattern

Builder Pattern adalah pola desain untuk membuat objek yang kompleks secara bertahap

Menggunakan method chaining yaitu memanggil method secara berantai dalam satu baris

Setiap method mengembalikan this agar bisa disambung ke method berikutnya

Diakhiri dengan method build() untuk menghasilkan objek final

Pengertian Extension Methods

Extension Methods adalah fitur untuk menambahkan method baru ke tipe yang sudah ada

Tidak perlu mengubah atau menyentuh kode asli dari tipe tersebut

Fitur ini tersedia sejak Dart versi 2.7

Analoginya seperti menambah kemampuan baru ke suatu benda tanpa mengubah benda aslinya

Masalah tanpa Extension Method

Sebelum adanya Extension Methods, penambahan fungsi pada tipe data biasanya dilakukan menggunakan utility class. Cara ini membuat penulisan kode terasa kurang natural karena fungsi tidak langsung menjadi bagian dari objek yang digunakan. Selain itu, programmer harus mengingat banyak class tambahan sehingga kode menjadi lebih panjang, kurang praktis, dan lebih sulit dibaca.

//Nama: Saskia Mauli Tantria
//NIM: 251410034
//Kelas: SI2A

// Utility class yang kurang natural
class StringUtils {
  static String capitalize(String text) {
    if (text.isEmpty) return text;
    return text[0].toUpperCase() + text.substring(1);
  }

  static bool isEmail(String text) {
    return text.contains('@');
  }
}

void main() {
  var name = 'john';
  var capitalized = StringUtils.capitalize(name); // 'John'
  var isEmail = StringUtils.isEmail('test@mail.com');

  print('Original name: $name');
  print('Capitalized name: $capitalized');
  print('Is "test@mail.com" an email? $isEmail');
}

Solusi dengan Extension Method

Ini adalah solusi dari slide sebelumnya. Dengan Extension Methods, bisa menambahkan fitur baru ke tipe String tanpa mengubah kode aslinya.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A
      
extension StringExtensions on String {
  String capitalize() {
    if (isEmpty) return this;
    return this[0].toUpperCase() + substring(1);
  }

  bool get isEmail => contains('@');

  String reverse() {
    return split('').reversed.join();
  }
}

void main() {
  var name = 'john';
  var capitalized = name.capitalize();
  var isEmail = 'test@mail.com'.isEmail;
  var reversed = 'hello'.reverse();

  print('Original name: $name');
  print('Capitalized name: $capitalized');
  print('Is "test@mail.com" an email? $isEmail');
  print('Reversed "hello": $reversed');
}

Sintaks Extension Method

Struktur dasar extension: extension NamaExtension on TipeTarget { }

Di dalam extension bisa berisi 3 jenis anggota:

Method → fungsi biasa yang dipanggil dengan tanda kurung ()

Getter → properti yang diakses tanpa tanda kurung, menggunakan kata kunci get

Operator → mendefinisikan ulang perilaku simbol seperti +, -, *

Kata kunci this di dalam extension merujuk ke objek yang memanggil method tersebut

// Nama: Saskia Mauli Tantria
// NIM: 251410034
// Kelas: SI2A

extension DateTimeExtensions on DateTime {
  // Method: format tanggal
  String toFormattedString() {
    return '$day/$month/$year';
  }

  // Getter: cek apakah hari ini
  bool get isToday {
    var now = DateTime.now();
    return year == now.year && month == now.month && day == now.day;
  }

  // Operator: tambah durasi
  DateTime operator +(Duration duration) {
    return add(duration);
  }
}

void main() {
  var today = DateTime.now();
  var someDay = DateTime(2020, 6, 15);
  var future = today + Duration(days: 7);

  print('Hari ini: ${today.toFormattedString()}');
  print('Tanggal lama: ${someDay.toFormattedString()}');
  print('Apakah today hari ini? ${today.isToday}');
  print('Apakah someDay hari ini? ${someDay.isToday}');
  print('7 hari ke depan: ${future.toFormattedString()}');
}

Tipe yang bisa di-Extend

Extension di Dart tidak terbatas hanya untuk String saja. Hampir semua tipe bisa di-extend, yaitu tipe bawaan Dart seperti String, int, double, List, Map, lalu kelas

buatan sendiri, generic types seperti List<int>, dan nullable types seperti String?. Ini membuat extension sangat fleksibel karena bisa disesuaikan dengan

kebutuhan tipe data apapun yang sedang digunakan.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension IntListExtensions on List<int> {
  int get sum => fold(0, (total, item) => total + item);
  double get average => isEmpty ? 0 : sum / length;
}

void main() {
  var numbers = [1, 2, 3, 4, 5];
  print(numbers.sum);      // 15
  print(numbers.average);  // 3.0
}

Extension untuk Null Safety

Nullable type artinya variabel bisa bernilai null

Tanpa extension, kita harus cek null secara manual setiap saat → kode jadi panjang dan berulang

Dengan extension pada String?, pengecekan null cukup dibuat sekali lalu bisa dipakai di mana saja

Method yang ditambahkan:

orEmpty() → kembalikan string kosong jika null

isNullOrEmpty → cek apakah null atau kosong

isNotNullOrEmpty → kebalikan dari isNullOrEmpty

withDefault('Guest') → kembalikan nilai cadangan jika null

Hasilnya kode lebih bersih, aman, dan mudah dibaca

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension NullableStringExtensions on String? {
  String orEmpty() => this ?? '';
  bool get isNullOrEmpty => this == null || this!.isEmpty;
  bool get isNotNullOrEmpty => !isNullOrEmpty;
  String withDefault(String defaultValue) => this ?? defaultValue;
}

void main() {
  String? name;
  print(name.orEmpty());             // ''
  print(name.isNullOrEmpty);         // true
  print(name.isNotNullOrEmpty);      // false
  print(name.withDefault('Guest'));  // Guest
}

Praktik 1- Utilities untuk Dart Types

Extension dibuat pada tipe num yang mencakup int dan double

toRupiah() → mengubah angka menjadi format mata uang Rupiah

isBetween() → mengecek apakah angka berada dalam rentang tertentu

power() → menghitung pangkat dari sebuah angka

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension NumberExtensions on num {
  String toRupiah() {
    return 'Rp${toStringAsFixed(0).replaceAllMapped(
      RegExp(r'(\d{1,3})(?=(\d{3})+(?!\d))'),
      (match) => '${match[1]}.',
    )}';
  }

  bool isBetween(num min, num max) {
    return this >= min && this <= max;
  }

  num power(int exponent) {
    var result = this;
    for (var i = 1; i < exponent; i++) {
      result *= this;
    }
    return result;
  }
}

void main() {
  var price = 1500000;
  print(price.toRupiah());        // Rp1.500.000
  print(5.isBetween(1, 10));      // true
  print(2.power(3));              // 8
}

Generic Extension

Generic extension <T> berarti bisa dipakai untuk List tipe apapun

getOrNull() → mengambil item di index tertentu, jika tidak ada kembalikan null (aman, tidak error)

chunk() → memecah list besar menjadi beberapa list kecil dengan ukuran tertentu

distinct() → menghapus data duplikat dalam list

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension ListExtensions<T> on List<T> {
  T? getOrNull(int index) {
    if (index < 0 || index >= length) return null;
    return this[index];
  }

  List<List<T>> chunk(int size) {
    var chunks = <List<T>>[];
    for (var i = 0; i < length; i += size) {
      chunks.add(sublist(i, i + size > length ? length : i + size));
    }
    return chunks;
  }

  List<T> distinct() {
    var result = <T>[];
    for (var item in this) {
      if (!result.contains(item)) {
        result.add(item);
      }
    }
    return result;
  }
}

void main() {
  var numbers = [1, 2, 3, 4, 5, 6, 7];
  print(numbers.chunk(3));               // [[1,2,3],[4,5,6],[7]]
  print([1, 2, 2, 3, 3, 3].distinct()); // [1, 2, 3]
  print(numbers.getOrNull(10));          // null
}

Unnamed Extensions

Extension tanpa nama dibuat dengan extension on TipeTarget tanpa menulis nama

Hanya bisa digunakan dalam file yang sama, tidak bisa di-import ke file lain

Cocok untuk helper kecil yang tidak perlu dipakai di luar file tersebut

Mencegah polusi namespace karena tidak terekspos ke luar

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension on DateTime {
  String get toYYYYMMDD {
    return '$year-${month.toString().padLeft(2, '0')}-${day.toString().padLeft(2, '0')}';
  }

  DateTime get startOfDay {
    return DateTime(year, month, day);
  }
}

void main() {
  var today = DateTime.now();
  print(today.toYYYYMMDD);   // contoh: 2026-05-19
  print(today.startOfDay);   // contoh: 2026-05-19 00:00:00.000
}

Konflik Nama Extension

Konflik terjadi jika dua extension punya nama method yang sama untuk tipe yang sama

Memanggil langsung seperti text.capitalize() akan menyebabkan error ambiguous

Solusi 1: Gunakan explicit extension call NamaExtension(object).method()

Solusi 2: Gunakan import ... as untuk memberi prefix, atau hide untuk menyembunyikan salah satu method

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension StringCaseExtensions on String {
  String capitalize() {
    if (isEmpty) return this;
    return this[0].toUpperCase() + substring(1);
  }
}

extension TextUtilsExtensions on String {
  String capitalize() {
    return toUpperCase(); // implementasi berbeda
  }
}

void main() {
  var text = 'hello';

  // Panggil eksplisit untuk hindari konflik
  print(StringCaseExtensions(text).capitalize()); // Hello
  print(TextUtilsExtensions(text).capitalize());  // HELLO
}

Extension untuk Custom Classes

Extension bisa dibuat untuk kelas buatan sendiri (custom class), bukan hanya tipe bawaan Dart

Memisahkan logika tambahan dari class utama agar class tetap bersih dan sederhana

withDiscount() → menambah fitur hitung diskon tanpa mengubah class Product

displayInfo → getter untuk menampilkan info produk secara terformat

isExpensive → getter untuk mengecek apakah produk termasuk mahal

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

class Product {
  final String name;
  final double price;
  Product(this.name, this.price);
}

extension DoubleRupiahFormatter on double {
  String toRupiah() {
    final intValue = toInt();
    String numberString = intValue.toString();
    String formattedString = '';
    int counter = 0;
    for (int i = numberString.length - 1; i >= 0; i--) {
      formattedString = numberString[i] + formattedString;
      counter++;
      if (counter % 3 == 0 && i != 0) {
        formattedString = '.' + formattedString;
      }
    }
    return 'Rp$formattedString';
  }
}

extension ProductExtensions on Product {
  Product withDiscount(double percentage) {
    var discountedPrice = price * (1 - percentage / 100);
    return Product(name, discountedPrice);
  }

  String get displayInfo => '$name - ${price.toRupiah()}';
  bool get isExpensive => price > 1000000;
}

void main() {
  var laptop = Product('Laptop', 15000000);
  var discounted = laptop.withDiscount(10);
  print(discounted.displayInfo);  // Laptop - Rp13.500.000
  print(laptop.isExpensive);      // true

  var keyboard = Product('Keyboard', 750000);
  print(keyboard.displayInfo);    // Keyboard - Rp750.000
  print(keyboard.isExpensive);    // false
}

Praktik 2 Validator Extensions

Extension digunakan untuk membuat validator yang bisa langsung dipanggil dari nilai itu sendiri

StringValidators menambahkan validasi pada String: cek email, nomor HP, password kuat, dan URL

IntValidators menambahkan validasi pada int: cek umur dewasa, positif, genap, dan ganjil

Semua validasi berbentuk getter sehingga penggunaannya simpel tanpa tanda kurung

Kode validasi menjadi lebih rapi karena tidak perlu class atau fungsi terpisah

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

extension StringValidators on String {
  bool get isEmail =>
      RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$').hasMatch(this);

  bool get isPhoneNumber =>
      RegExp(r'^[0-9]{10,13}$').hasMatch(this);

  bool get isStrongPassword =>
      length >= 8 &&
      contains(RegExp(r'[A-Z]')) &&
      contains(RegExp(r'[a-z]')) &&
      contains(RegExp(r'[0-9]'));

  bool get isUrl {
    try {
      Uri.parse(this);
      return true;
    } catch (_) {
      return false;
    }
  }
}

extension IntValidators on int {
  bool get isAdultAge => this >= 18;
  bool get isPositive => this > 0;
  bool get isEven => this % 2 == 0;
  bool get isOdd => this % 2 != 0;
}

void main() {
  var email = 'test@example.com';
  var age = 25;
  var password = 'Pass1234';

  print(email.isEmail);             // true
  print(age.isAdultAge);            // true
  print(password.isStrongPassword); // true
  print(10.isEven);                 // true
}

Kesimpulan Poin Penting

Ringkasan

- Extension methods digunakan untuk menambah method ke tipe yang sudah ada tanpa mengubah kode aslinya

- Sintaks dasarnya: `extension Name on Type { }`

- Di dalamnya bisa berisi method, getter, setter, dan operator

- Bisa digunakan untuk generic types seperti `List<T>`

Best Practices

- Nama jelas → beri nama extension yang deskriptif seperti `StringExtensions` atau `DateTimeUtils`

- Group by type → satu file khusus untuk satu tipe, agar kode lebih terorganisir

- Avoid overuse → tidak semua fungsi perlu dijadikan extension, gunakan seperlunya

- Null safety → tangani nullable types dengan baik agar kode aman dari error null

- Dokumentasi → tambahkan komentar pada extension yang bersifat publik agar mudah dipahami orang lain
